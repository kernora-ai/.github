# Kernora

**The AI work intelligence layer for developers.**

Kernora builds [Nora](https://github.com/kernora-ai/nora) — an open-source AI work intelligence layer that runs alongside your AI coding tools (Claude Code, Cursor, Kiro, VS Code), captures every session, learns your patterns, and feeds knowledge back into your next session. Every session makes the next one smarter.

## What lives here

- **[`nora`](https://github.com/kernora-ai/nora)** — the open-source Nora client (MCP server + dashboard + CLI). Local-first, no cloud required.

We also maintain the [Factlet Protocol](https://github.com/factlet-ai) — an open specification for portable, vendor-neutral AI grounding that Nora implements.
